## 📋 项目概述

**习惯养成**是一款亲子任务奖励应用：
- 孩子完成任务 → 获得星星 → 兑换愿望奖励
- 目标用户：3-12岁儿童及家长
- 技术栈：Vue3 + Koa2 + PostgreSQL

## ⚙️ 后端开发

### 路由定义
```javascript
// backend/src/routes/tasks.js
router.get('/', getTasks)           // GET /api/tasks
router.post('/', createTask)        // POST /api/tasks
router.put('/:id', updateTask)     // PUT /api/tasks/:id
```

### 添加新 API 步骤
1. 在 `controllers/` 添加控制器函数
2. 在 `routes/` 添加路由
3. 前端 `api/` 添加请求封装
4. 更新 `docs/api.md`

### 积分操作（重要！）
所有积分操作必须：
1. 记录到 `point_logs` 表
2. 同时更新 `users.stars` 和 `users.total_stars`
3. 在事务中完成
4. 使用 `FOR UPDATE` 锁定

```javascript
// ✅ 正确示例
await client.query('BEGIN')
await client.query(
  'UPDATE users SET stars = stars + $1, total_stars = total_stars + $1 WHERE id = $2',
  [amount, userId]
)
await client.query(
  'INSERT INTO point_logs (user_id, amount, type, description) VALUES ($1, $2, $3, $4)',
  [userId, amount, 'task_approve', '完成任务获得星星']
)
await client.query('COMMIT')
```

## 🐛 常见问题

### 1. API 404 错误
检查：
- 前端 baseURL 是否正确（`/api`）
- 路由是否正确挂载到 `app.js`
- 路径是否有重复 `/api/api/`

### 2. 积分数据不一致
运行修复脚本：
```bash
/opt/xuexi-app/scripts/fix_points_consistency.sh
```

### 3. 时区问题
确保数据库和服务器都使用 CST 时区：
```sql
ALTER DATABASE xuexi SET timezone = 'Asia/Shanghai';
```

### 4. 前端样式异常
清除缓存：
```bash
rm -rf frontend/node_modules/.vite
```

```bash
tail -f /tmp/xuexi-frontend.log

# 积分一致性核查日志
cat /opt/xuexi-app/logs/points_consistency.log
```

### 数据库操作
```bash
# 连接数据库
docker exec -it xuexi-postgres psql -U postgres -d xuexi

# 查看用户积分
SELECT nickname, stars, total_stars FROM users WHERE role = 'child';

# 查看积分日志
SELECT * FROM point_logs ORDER BY created_at DESC LIMIT 10;

# 核查积分一致性
SELECT u.id, u.nickname, u.stars, 
       COALESCE(SUM(pl.amount), 0) as logs_balance
FROM users u LEFT JOIN point_logs pl ON u.id = pl.user_id
WHERE u.role = 'child'
GROUP BY u.id, u.nickname, u.stars;
```
