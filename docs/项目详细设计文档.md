# 习惯养成 App - 项目技术文档

> 孩子完成任务 → 获得星星 → 兑换愿望

---

## 一、项目概述

### 1.1 产品定位
- **目标用户**：3-12岁儿童及其家长
- **核心功能**：通过游戏化任务系统培养孩子良好习惯
- **核心玩法**：完成任务获得星星 → 积累星星兑换愿望奖励

### 1.2 主要特性
- 📱 移动端优先设计
- 🎮 游戏化激励机制（等级、稀有度、排行榜）
- 👨‍👩‍👧‍👦 亲子互动（家庭管理、多孩子支持）
- ⭐ 星星货币系统
- 🏆 成就系统

### 4.2 表结构详解

#### 4.2.1 家庭表 (family)

```sql
CREATE TABLE family (
    id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name        VARCHAR(100) NOT NULL DEFAULT '我的家庭',
    code        VARCHAR(20) UNIQUE NOT NULL,  -- 邀请码，用于加入家庭
    owner_id    UUID REFERENCES users(id),       -- 家长用户ID
    created_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

**说明：**
- `code`：6位随机字符串，家庭成员可通过邀请码加入
- `owner_id`：创建家庭的家长用户

#### 4.2.2 用户表 (users)

```sql
CREATE TABLE users (
    id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    family_id   UUID REFERENCES family(id),      -- 所属家庭
    openid      VARCHAR(100) UNIQUE,             -- 设备ID/微信openid
    nickname    VARCHAR(50) NOT NULL,            -- 昵称
    avatar      VARCHAR(500) DEFAULT '',           -- 头像URL
    role        VARCHAR(20) DEFAULT 'child',       -- 角色：parent/child
    level       INTEGER DEFAULT 1,                -- 等级
    stars       INTEGER DEFAULT 0,                -- 当前星星数
    wish_points INTEGER DEFAULT 0,                 -- 愿望值（累计）
    is_current  BOOLEAN DEFAULT FALSE,             -- 是否当前选中的孩子
    created_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

**说明：**
- `role`：区分家长和孩子，家长可管理设置，孩子可完成任务
- `stars`：可用于兑换的星星余额
- `wish_points`：累计获得的星星数（不扣除）

#### 4.2.3 目标表 (goals)

```sql
CREATE TABLE goals (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    family_id       UUID REFERENCES family(id) NOT NULL,
    user_id         UUID REFERENCES users(id),   -- 指定孩子，NULL表示通用
    title           VARCHAR(100) NOT NULL,        -- 目标名称，如"坚持阅读"
    icon            VARCHAR(50) DEFAULT 'star',  -- 图标
    difficulty      INTEGER DEFAULT 10,           -- 难度：10(轻松)/50(普通)/100(挑战)
    star_target     INTEGER DEFAULT 10,            -- 需要达成的星星数
    current_stars   INTEGER DEFAULT 0,            -- 当前累计星星
    status          VARCHAR(20) DEFAULT 'active',  -- active/completed/abandoned
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

**说明：**
- `difficulty`：对应 10★=3-5天，50★=1-2周，100★=3-4周
- `user_id`：NULL时表示所有孩子共享此目标

#### 4.2.4 任务表 (tasks)

```sql
CREATE TABLE tasks (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    family_id       UUID REFERENCES family(id) NOT NULL,
    goal_id         UUID REFERENCES goals(id),   -- 关联目标
    title           VARCHAR(100) NOT NULL,        -- 任务名称
    icon            VARCHAR(50) DEFAULT 'todo-o', -- 图标
    star_reward     INTEGER DEFAULT 1,            -- 完成奖励星星
    rarity          VARCHAR(20) DEFAULT 'N',     -- N/R/SR/SSR 稀有度
    frequency       VARCHAR(20) DEFAULT 'daily', -- daily/weekly 频率
    frequency_count INTEGER DEFAULT 1,             -- 每日/每周次数
    is_active       BOOLEAN DEFAULT TRUE,         -- 是否启用
    sort_order     INTEGER DEFAULT 0,             -- 排序
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

**稀有度说明：**
| 稀有度 | 颜色 | 说明 |
|--------|------|------|
| N | 灰色 | 普通任务 |
| R | 蓝色 | 稀有任务 |
| SR | 紫色 | 超稀有任务 |
| SSR | 橙色 | 极限任务 |

#### 4.2.5 奖励表 (rewards)

```sql
CREATE TABLE rewards (
    id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    family_id   UUID REFERENCES family(id) NOT NULL,
    title       VARCHAR(100) NOT NULL,            -- 奖励名称
    icon        VARCHAR(50) DEFAULT 'gift',        -- 图标
    star_cost   INTEGER NOT NULL,                  -- 所需星星
    rarity      VARCHAR(20) DEFAULT 'normal',      -- normal/epic/legend
    is_active   BOOLEAN DEFAULT TRUE,              -- 是否启用
    sort_order  INTEGER DEFAULT 0,                -- 排序
    created_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

#### 4.2.6 任务记录表 (task_logs)

```sql
CREATE TABLE task_logs (
    id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id     UUID REFERENCES users(id) NOT NULL,
    task_id     UUID REFERENCES tasks(id) NOT NULL,
    action      VARCHAR(20) NOT NULL,              -- complete/skip
    stars_earned INTEGER DEFAULT 0,                 -- 获得的星星
    created_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

**说明：**
- `action='complete'`：完成任务，获得星星
- `action='skip'`：跳过任务，不获得星星

#### 4.2.7 兑换记录表 (exchange_logs)

```sql
CREATE TABLE exchange_logs (
    id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id     UUID REFERENCES users(id) NOT NULL,
    reward_id   UUID REFERENCES rewards(id) NOT NULL,
    stars_spent INTEGER NOT NULL,                   -- 消耗的星星
    status      VARCHAR(20) DEFAULT 'pending',     -- pending/completed/cancelled
    created_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

#### 4.2.8 成就表 (achievements)

```sql
CREATE TABLE achievements (
    id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id     UUID REFERENCES users(id) NOT NULL,
    type        VARCHAR(50) NOT NULL,              -- goal_completed/task_count/streak
    title       VARCHAR(100),                       -- 成就名称
    count       INTEGER DEFAULT 0,                 -- 完成次数
    unlocked_at TIMESTAMP,                         -- 解锁时间
    created_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 五、API 接口设计

### 5.1 接口基础信息

- **Base URL**: `http://your-domain.com/api`
- **认证方式**: Bearer Token (JWT)
- **Content-Type**: `application/json`
- **字符编码**: UTF-8

### 5.2 统一响应格式

```json
// 成功响应
{
  "code": 0,
  "message": "success",
  "data": { ... }
}

// 错误响应
{
  "code": 1001,
  "message": "错误描述",
  "data": null
}
```

### 5.3 错误码定义

| 错误码 | 说明 |
|--------|------|
| 0 | 成功 |
| 1001 | 参数错误 |
| 1002 | 未登录 |
| 1003 | 无权限 |
| 2001 | 用户不存在 |
| 2002 | 家庭不存在 |
| 3001 | 目标不存在 |
| 3002 | 任务不存在 |
| 3003 | 星星不足 |
| 3004 | 奖励不存在 |
| 500 | 服务器内部错误 |
