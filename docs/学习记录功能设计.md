# 学习记录功能设计文档

> 最后更新: 2026-03-29
> 版本: v1.0

---

## 1. 功能概述

### 1.1 核心流程

```
家长拍照试卷 → AI 识别题目 → AI 生成答案 → 家长审核校正 → 入题库
                                                                      ↓
孩子按科目练习 ← 随机抽题 ← 做错的题自动入错题本
              ↓
         家长可查看学习记录和错题本
```

### 1.2 角色权限

| 角色 | 可操作 |
|------|--------|
| 家长 | 拍照登记、审核题目、管理科目、查看统计、做任何操作 |
| 孩子 | 做练习、查看错题本、标记掌握 |

## 2. 数据库设计

### 2.1 表结构

```sql
-- ============================================
-- 科目表
-- ============================================
CREATE TABLE study_subjects (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    family_id UUID NOT NULL REFERENCES family(id),
    name VARCHAR(50) NOT NULL,           -- 科目名称：语文、数学、英语
    icon VARCHAR(20),                     -- emoji图标
    color VARCHAR(20),                    -- 主题色
    sort_order INT DEFAULT 0,
    is_active BOOLEAN DEFAULT true,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    UNIQUE(family_id, name)
);

-- ============================================
-- 学习记录表
-- ============================================
CREATE TABLE study_records (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL REFERENCES users(id),  -- 家长ID
    subject_id UUID NOT NULL REFERENCES study_subjects(id),
    record_date DATE NOT NULL,             -- 记录日期
    photos JSONB DEFAULT '[]',           -- 试卷照片路径数组
    notes TEXT,                           -- 家长备注
    status VARCHAR(20) DEFAULT 'draft',  -- draft/approved
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- ============================================
-- 题目表
-- ============================================
CREATE TABLE study_questions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    subject_id UUID NOT NULL REFERENCES study_subjects(id),
    record_id UUID REFERENCES study_records(id),  -- 来源记录
    question_no VARCHAR(20) NOT NULL,    -- 题号：1、2、3 或 1(a)、2(b)
    question_type VARCHAR(20) NOT NULL, -- choice/truefalse/fill/blank/application
    content TEXT NOT NULL,              -- 题目内容（主干）
    options JSONB,                     -- 选择题选项 [{"label":"A","text":"..."}]
    ai_answer TEXT,                   -- AI生成的答案
    correct_answer TEXT,               -- 家长校正后的正确答案
    answer_image VARCHAR(500),          -- 答案页照片路径（可选）
    parent_reviewed BOOLEAN DEFAULT false,  -- 家长是否已审核
    difficulty VARCHAR(20) DEFAULT 'medium', -- easy/medium/hard
    times_asked INT DEFAULT 0,        -- 被抽中次数
    times_correct INT DEFAULT 0,      -- 答对次数
    source VARCHAR(20) DEFAULT 'photo', -- photo/manual
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- ============================================
-- 错题本表
-- ============================================
CREATE TABLE study_wrong_questions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL REFERENCES users(id),  -- 孩子ID
    question_id UUID NOT NULL REFERENCES study_questions(id),
    times_wrong INT DEFAULT 1,         -- 做错次数
    times_reviewed INT DEFAULT 0,     -- 复习次数
    mastered BOOLEAN DEFAULT false,   -- 已掌握
    last_reviewed TIMESTAMPTZ,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    UNIQUE(user_id, question_id)
);

-- ============================================
-- 练习记录表
-- ============================================
CREATE TABLE study_practice_logs (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL REFERENCES users(id),
    subject_id UUID NOT NULL REFERENCES study_subjects(id),
    question_ids JSONB NOT NULL,      -- 本次练习的题目ID数组
    answers JSONB NOT NULL,           -- 孩子提交的答案 {"question_id": "answer"}
    results JSONB NOT NULL,           -- 判定结果 {"question_id": true/false}
    score INT,                       -- 正确率百分比
    practiced_at TIMESTAMPTZ DEFAULT NOW()
);

-- ============================================
-- 复习记录表
-- ============================================
CREATE TABLE study_review_logs (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL REFERENCES users(id),
    question_id UUID NOT NULL REFERENCES study_questions(id),
    result VARCHAR(20) NOT NULL,      -- correct/wrong
    reviewed_at TIMESTAMPTZ DEFAULT NOW()
);
```

### 2.2 索引

```sql
CREATE INDEX idx_records_user_date ON study_records(user_id, record_date);
CREATE INDEX idx_records_subject ON study_records(subject_id);
CREATE INDEX idx_questions_subject ON study_questions(subject_id);
CREATE INDEX idx_questions_record ON study_questions(record_id);
CREATE INDEX idx_wrong_user ON study_wrong_questions(user_id);
CREATE INDEX idx_practice_user ON study_practice_logs(user_id, practiced_at);
```

## 3. API 设计

### 3.1 基础信息

- **基础 URL**: `/api/study`
- **认证**: Bearer Token (JWT)
- **支持角色**: 家长(`admin`)、孩子(`child`)

### 3.2 科目管理

```
GET    /api/study/subjects           # 获取科目列表
POST   /api/study/subjects           # 创建科目（家长）
PUT    /api/study/subjects/:id       # 更新科目（家长）
DELETE /api/study/subjects/:id       # 删除科目（家长）
```

**POST /api/study/subjects**
```json
// Request
{ "name": "数学", "icon": "📐", "color": "#4A90D9" }

// Response
{ "code": 0, "data": { "id": "...", "name": "数学", ... } }
```

### 3.3 学习记录

```
GET    /api/study/records            # 学习记录列表（家长看全部，孩子只看自己的）

PUT    /api/study/records/:id        # 更新记录（家长）
DELETE /api/study/records/:id        # 删除记录（家长）
```

## 5. OCR + AI 答案生成

### 5.1 流程

```
1. 家长拍照上传
2. 前端压缩图片 (压缩率 0.7, 最大 1MB)
3. POST /api/study/ocr
4. 后端调用 MiniMax Vision 识别图片文本
5. 后端解析文本 → 按题号切分
6. 后端调用 MiniMax 生成每题答案
7. 返回题目列表
8. 家长逐题审核、校正答案
9. 确认后批量入库
```

### 5.2 题号识别正则

```javascript
// 题号切分模式
const patterns = [
  /^(第?\d+[题个]|[一二三四五六七八九十]+[题个])/i,  // 第1题、1题、一题
  /^(\d+)[\.、\.．]/,                                 // 1. 2. 3.
  /^【(\d+)】/,                                       // 【1】
  /^\((\d+)\)/,                                     // (1)
  /^[①-⑳]/,                                        // ①②③
]
```

### 5.3 题型识别

```javascript
// 从题目内容识别题型
const detectType = (content) => {
  if (/选择|下列|哪个/.test(content)) return 'choice'
  if (/正确|错误|对不对|错对/.test(content)) return 'truefalse'
  if (/填.*空|____|______/.test(content)) return 'fill'
  if (/解答|应用|计算|证明/.test(content)) return 'application'
  return 'choice' // 默认选择题
}
```

## 8. 注意事项

1. **照片压缩**: 前端必须压缩后再上传，建议使用 `compress-formatimage` 库
2. **OCR准确率**: 手写体识别有限，必须有家长校正环节
3. **错题入库时机**: 练习提交后自动比对答案，答错自动入错题本
4. **掌握判定**: 同一题连续答对3次自动移出错题本
5. **照片清理**: 删除学习记录时同时删除关联的照片文件
